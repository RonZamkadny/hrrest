## описание запросов API
* GET http://localhost:8080/employee/all - вернуть всех сотрудников
* GET http://localhost:8080/employee/fire/1 - уволить сотрудника с табельным номером 1
* POST localhost:8080/employee/hire - нанять сотрудника
* POST http://localhost:8080/employee/updateProfile/1 - обновить профиль сотрудника с номером 1
* GET http://localhost:8080/employee/get/1 - получить информацию о сотруднике
